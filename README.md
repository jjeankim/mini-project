# 경주 문화 관광 사이트


<br />


## 📄프로젝트 소개
경주 문화 관광 사이트는 공공 데이터 포털의 open API를 활용하여 경주 관광을 위한 먹거리 핫플레이스, 카페, 관광지를 확인할 수 있습니다.
React 기반 SPA로 구성되어 있습니다.
