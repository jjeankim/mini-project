## 📄프로젝트 소개
경주 문화 관광 사이트는 공공 데이터 포털의 open API를 활용하여 경주 관광을 위한 테마별 숙소, 먹거리 핫플레이스, 카페 및 전통 찻집, 권역별 관광지를 확인할 수 있습니다.
React 기반 SPA로 구성되어 있습니다.

## 🧑‍💻프로젝트 주요 기능

| 기능           | 설명                                                                              |
|----------------|-----------------------------------------------------------------------------------|
| 카테고리별 정보 확인 | 사용자가 원하는 카테고리별(예: 음식, 관광지 등)에 따라 관광 정보를 조회할 수 있습니다.|
| 일기 예보 확인 | 오늘, 내일, 모레의 날씨 정보를 확인하여 여행 일정을 미리 계획할 수 있습니다.|
| 무한 스크롤 | 사용자가 스크롤을 내릴 때마다 자동으로 다음 콘텐츠가 로딩되어 자연스럽게 많은 정보를 확인할 수 있습니다.|
| 페이지네이션 | 페이지 번호를 통해 데이터를 구간별로 나누어 조회할 수 있어, 원하는 구간만 빠르게 찾아볼 수 있습니다.|
| 필터링 | 권역별 관광지를 선택하여 정보를 확인할 수 있습니다.|
| 상세 정보 확인 | 선택한 관광지나 항목에 대해 자세한 설명, 사진, 위치 등 추가 정보를 확인할 수 있습니다.|
